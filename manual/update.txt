How to deploy those modifications on the server
Because you already made some hotfixes directly on the VPS, do this safely.

1. Pull your new commit
cd /home/ubuntu/Photographic-Portfolio-Manager
git pull

2. Rebuild backend
Use this form even if mvnw executable bit is still not fixed:
cd /home/ubuntu/Photographic-Portfolio-Manager/backend
bash ./mvnw clean package

3. Redeploy backend jar
sudo cp target/backend-0.0.1-SNAPSHOT.jar /opt/Photographic-Portfolio-Manager/backend.jar
sudo chown ubuntu:ubuntu /opt/Photographic-Portfolio-Manager/backend.jar
sudo systemctl restart photographic-portfolio-manager
sudo systemctl status photographic-portfolio-manager --no-pager -l

4. Rebuild and publish frontend
cd /home/ubuntu/Photographic-Portfolio-Manager/frontend
npm ci
VITE_API_BASE_URL="" npm run build
sudo rsync -a --delete dist/ /var/www/letmelens/

5. Reload Nginx
sudo nginx -t
sudo systemctl reload nginx

6. Verify
curl -i http://127.0.0.1:8080/api/auth/me
curl -I http://www.letmelens.com
curl -I https://www.letmelens.com
sudo systemctl status photographic-portfolio-manager --no-pager -l
